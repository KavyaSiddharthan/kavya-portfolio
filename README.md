## Installation  
1. **Clone the repository**  
```bash
git clone https://github.com/KavyaSiddharthan/kavya_portfolio.git
```

2. **Install dependencies**

```bash
npm install
```

3. **Run the project**

* Development:

```bash
npm run dev
```

* Production:

```bash
npm start
```
